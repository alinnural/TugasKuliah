Node pada jaringan : mesin atau komputer yang menyediakan koneksi bagi user dalam kasus circuit switching atau merelay data user dalam kasus komunikasi data
Node bisa berupa : customer, switching,transmission, service

WAN (Wide) : Circuit switching, packet switching, frame relay, ATM (asynchronous Transfer Mode)
Circuit Switching : jalur komunikasi khusus dibangun selama durasi komunikasi, contoh : jaringan telepon, ISDN (integrated service digital network)
3 fase : establisment, data transfer, termination

Packet Switching : data terkirim dalam bentuk paket, potongan kecil dalam satu waktu. Melintas antar node dan digunakan untuk komunikasi antar komputer atau terminal. contoh : TCP/IP, ATM, GPRS, GSM.
2 tipe paket : Connectionless packet switching, dan Virtual Circuit Switching

Frame Relay : menerapkan virtual circuit, mengatasi overhead sistem packet switching (packet error), menyediakan koneksi kecepatan tinggi

ATM : evolusi dari frame relay, sedikit overhead untuk error control, ukuran packet tetap, kecepatan 10Mbps - Gbps, constant data rate dnegan mengunakan packet switching technique

Protokol Jaringan
Aturan pikah yang berkomunikasi dan berinteraksi agar memberikan berbagai jenis layanan.

Fungsi jaringan :
switching : interkoneksi kanal
routing : memilih lintasan dengan syarat tertentu
flow control : memperlambat traffic
security : mencegah akses
backup : bereaksi terhadap kegagalan
monitoring : traffic akuntabilitas
internetworking : menyediakan komunikasi dengan jaringan lain
manajemen jaringan : pengelolaan resource jaringan

Kinerja Jaringan
Delay/latency : processing delay, queuing delay, transmission delay
Througput
Packet loss
Based on service requirement
QoS pada jaringan multimedia : delay sensitif (audio 150 ms), loss tolerant

Transport Layer -> RTP
